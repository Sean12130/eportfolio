Professional Self-Assessment

Throughout my Computer Science program, I have developed a strong foundation in software development, algorithms, databases, and secure system design. Completing courses ranging from introductory scripting and discrete mathematics to advanced topics such as software security, full-stack development, and system architecture has helped me build both technical depth and practical problem-solving skills. My ePortfolio reflects this growth by showcasing projects that demonstrate my ability to design, implement, test, and improve software systems in real-world contexts.

One of the most important skills I developed during the program is the ability to work effectively in collaborative and professional environments. Across multiple courses, including CS 250 Software Development Lifecycle and CS 340 Client/Server Development, I learned how software is built through structured processes and team-based decision making. My code review experience further strengthened my ability to evaluate software critically and communicate feedback clearly. This helped me understand how collaboration improves code quality and supports better system design. I also developed stronger communication skills by explaining technical ideas through written, visual, and verbal formats for different audiences and stakeholders.

Another major area of growth was in data structures and algorithms. In courses such as CS 210 Programming Languages and CS 300 Data Structures and Algorithms, I learned how to design efficient solutions using maps, lists, sorting, and searching techniques. My ItemTracker project demonstrates this by processing large datasets and calculating item frequencies using structured data storage and sorting algorithms. This experience helped me understand how algorithmic decisions impact performance, scalability, and usability in software systems.

In software engineering and database development, I gained hands-on experience building full applications that integrate multiple components. In CS 340 Client/Server Development, I developed an Animal Shelter Dashboard using Python, Dash, and MongoDB. This project required connecting a database to an interactive interface and ensuring all components worked together correctly. I learned how important it is to structure software properly, manage data flow, and design systems that are both functional and user-friendly.

Security and software quality were also key parts of my development. In courses such as CS 305 Software Security and CS 405 Secure Coding, I developed a security mindset focused on input validation, vulnerability prevention, and designing systems that reduce risk. This influenced my approach to validating user input and preventing invalid or duplicate data in my projects. I also learned to consider how future changes in software design could introduce vulnerabilities if not properly planned.

My ePortfolio brings all of these skills together into a cohesive representation of my abilities as a software developer. Each artifact highlights a different area of my learning, including algorithms and data structures, software engineering, and database integration. Together, they demonstrate my ability to design and evaluate computing solutions, manage trade-offs in system design, and apply best practices in software development.
